# Thinreports::CLI

**Unofficial** and **Experimental** Thinreports command-line tool.

## Installation

    $ gem install thinreports-cli --pre

## Commands

### $ thinreports upgrade

Upgrade .tlf schema to 0.9.x from 0.8.x:

```
$ thinreports upgrade /path/to/old-0.8.x.tlf /path/to/new-0.9.x.tlf
```

## Requirements

 - Ruby 2.0.0+
 - thinreports 0.9.x
